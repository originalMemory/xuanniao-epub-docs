本软件目的是为了快速从小说 txt 拆分章节制作 epub 。使用时不需要关注文件和目录的映射关系，简化元数据配置复杂度，不需要手动调整很多东西。

本软件可以根据规则自动拆分章节，并随意拖拽调整章节顺序和层级。有可编辑的默认样式，支持上传或粘贴封面，可以拖拽、粘贴、从已有图片中选择插入图片到当前章节的任意位置。

> 其实就是因为 Sigil 用得太不顺手了。它是非常全面的 epub 编辑器，完整还原 epub 格式，可以实现很多高级操作。但是对于简单的需求，比如把一本小说的 txt 拆分章节，补充元数据信息和封面生成 epub 。用 Sigil 步骤是实现是太多了。

当前仅有 windows 版，后续支持 mac 版。
